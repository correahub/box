# `fr` é a unidade de medida que representa **uma fração do espaço disponível do container**.




















No CSS (especialmente no **Grid Layout**), a unidade `fr` significa **“fração do espaço disponível”** 📐.

Ela é usada para dividir o espaço restante de um container de forma proporcional.

---

## 💡 O que é `1fr`?

`1fr` = **1 parte do espaço disponível**

Ou seja, depois que o navegador calcula tudo que tem tamanho fixo (como `px`, `%`, `auto`, etc.), o que sobra é distribuído entre os elementos que usam `fr`.

---

## 🔹 Exemplo simples

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr;
}
```

👉 Aqui:

- O espaço disponível é dividido em **2 partes iguais**
    
- Cada coluna ocupa **50% do espaço**
    

---

## 🔹 Exemplo com proporção

```css
.container {
  display: grid;
  grid-template-columns: 1fr 2fr;
}
```

👉 Aqui:

- Total de frações = 3 (1 + 2)
    
- Primeira coluna = **1/3**
    
- Segunda coluna = **2/3**
    

---

## 🔹 Misturando com valores fixos

```css
.container {
  display: grid;
  grid-template-columns: 200px 1fr;
}
```

👉 Aqui:

- A primeira coluna tem **200px fixos**
    
- O restante do espaço vai para a segunda (`1fr`)
    

---

## 🧠 Resumindo

- `fr` divide **o espaço que sobra**, não o total
    
- `1fr` = uma fração desse espaço
    
- Pode usar várias frações para criar proporções (tipo 1:2:3)
