# **`flex-direction`** é o **atributo** que define **a direção em que os itens de um contêiner flexível são distribuídos**, podendo ser `row`, `row-reverse`, `column` ou `column-reverse`, decidindo assim por consequência o eixo secundário.
























---

## 📐 As direções possíveis

### 1️⃣ **row**

➡️ Os itens ficam **em linha**, da esquerda para a direita (padrão).  
É como escrever no caderno: você começa da esquerda e vai para a direita.

### 2️⃣ **row-reverse**

⬅️ Os itens ficam em linha, **mas invertidos** (da direita para a esquerda).

### 3️⃣ **column**

⬇️ Os itens ficam **em coluna**, de cima para baixo.  
É como empilhar blocos.

### 4️⃣ **column-reverse**

⬆️ Os itens ficam em coluna, **mas invertidos** (de baixo para cima).

---

## 🧸 Exemplo simples

```css
.container {
  display: flex;
  flex-direction: row; /* ou column, row-reverse, column-reverse */
}
```

---

## 📘 Definição técnica simples (como você pediu)

**flex-direction** define o eixo principal do container flex e a orientação na qual os itens flex serão organizados. 🎯📦✨
