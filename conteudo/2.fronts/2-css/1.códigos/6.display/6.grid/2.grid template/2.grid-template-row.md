# **`Grid-template-rows`** é usado para **definir a quantidade e a altura das linhas** dentro de um container com  grid.
























Ela é basicamente o equivalente vertical do `grid-template-columns`.

---

### 📌 Exemplo:

```css
.container {
  display: grid;
  grid-template-rows: 100px 1fr 50px;
}
```

### 🧠 O que isso significa:

- **100px** → primeira linha com altura fixa
    
- **1fr** → segunda linha ocupa o espaço restante
    
- **50px** → terceira linha com altura fixa
    

---

### 💡 Usando `fr`:

```css
grid-template-rows: 1fr 2fr;
```

👉 A segunda linha será **duas vezes maior** que a primeira

---

### 🧩 Usando `repeat`:

```css
grid-template-rows: repeat(3, 1fr);
```

👉 Cria **3 linhas iguais automaticamente**

---

### 📎 Resumo:

`grid-template-rows` serve para **controlar a altura e quantidade das linhas** em um layout grid.

---

Se quiser, posso juntar `columns` + `rows` e te mostrar como montar um layout completo (tipo página de site) 🚀
