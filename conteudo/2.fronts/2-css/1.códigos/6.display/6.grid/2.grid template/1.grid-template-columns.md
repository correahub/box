# O atributo **`grid-template-columns define quantas colunas existem e qual é o tamanho de cada uma dentro do container.






















Ela funciona junto com `display: grid`.

---

### 📌 Exemplo:

```css
.container {
  display: grid;
  grid-template-columns: 200px 1fr 100px;
}
```

### 🧠 O que isso significa:

- **200px** → primeira coluna fixa
    
- **1fr** → segunda coluna ocupa o espaço restante
    
- **100px** → terceira coluna fixa
    

---

### 💡 Unidade importante:

- `fr` = “fração do espaço disponível”  
    Exemplo:
    

```css
grid-template-columns: 1fr 2fr;
```

👉 A segunda coluna será o **dobro do tamanho da primeira**

---

### 🧩 Outro exemplo comum:

```css
grid-template-columns: repeat(3, 1fr);
```

👉 Cria **3 colunas iguais automaticamente**

---

### 📎 Resumo:

`grid-template-columns` serve para **controlar a largura e quantidade das colunas** de um layout em grid.
