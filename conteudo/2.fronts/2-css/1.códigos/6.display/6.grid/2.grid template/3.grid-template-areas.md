# O grid template areas é usado para definir áreas das grades, usando nomes.

























---

### 📌 Exemplo:

```css
.container {
  display: grid;
  grid-template-areas:
    "header header"
    "menu   content"
    "footer footer";
}
```

### 🧠 O que isso faz:

Você cria um “mapa” da página:

- `"header header"` → ocupa duas colunas
    
- `"menu content"` → divide em duas áreas
    
- `"footer footer"` → ocupa tudo embaixo
    

---

### 📦 Ligando os elementos às áreas:

```css
.header { grid-area: header; }
.menu { grid-area: menu; }
.content { grid-area: content; }
.footer { grid-area: footer; }
```

---

### 💡 Resultado:

Você monta o layout como se fosse um desenho:

```
HEADER  HEADER
MENU    CONTENT
FOOTER  FOOTER
```

---

### ⚠️ Regras importantes:

- Os nomes devem ser **iguais** entre o template e os elementos
    
- Use `.` para espaços vazios:
    

```css
grid-template-areas:
  "header header"
  ".      content";
```

---

### 📎 Resumo:

`grid-template-areas` permite **organizar o layout de forma visual e semântica**, deixando o código mais fácil de entender.
