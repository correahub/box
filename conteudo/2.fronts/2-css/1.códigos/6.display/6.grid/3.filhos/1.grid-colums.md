# `grid-column` é o atributo que define a linha de início e a linha de fim que o elemento ocupará nas colunas.



















A propriedade `grid-column` no CSS é usada para **definir em quais colunas de um Grid um elemento deve ficar**, controlando sua posição horizontal.

---

## 💡 Definição direta

O atributo`grid-column` define **a linha de início e a linha de fim que um elemento ocupará nas colunas do grid**.

---

## 🔹 Exemplo básico

```css
.item {
  grid-column: 1 / 3;
}
```

👉 O elemento:

- começa na linha de coluna **1**
    
- termina na linha **3**
    
- ocupa **2 colunas**
    

---

## 🔹 Usando `span`

```css
.item {
  grid-column: span 2;
}
```

👉 O elemento ocupa **2 colunas**, a partir de onde ele começa.

---

## 🧠 Resumindo

- Controla **posição horizontal** no Grid
    
- Usa **linhas de coluna** como referência
    
- Pode definir início/fim ou apenas quantas colunas ocupar
