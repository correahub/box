# `grid-row`  é o atributo que define  a linha de início e a linha de fim que um elemento ocupará nas linhas.



















A propriedade `grid-row` no CSS é usada para **definir em quais linhas de um Grid um elemento deve ficar**, controlando sua posição vertical.

---

## 💡 Definição direta

`grid-row` é uma propriedade do CSS Grid que especifica **a linha de início e a linha de fim que um elemento ocupará nas linhas do grid**.

---

## 🔹 Exemplo básico

```css
.item {
  grid-row: 1 / 3;
}
```

👉 O elemento:

- começa na linha **1**
    
- termina na linha **3**
    
- ocupa **2 linhas**
    

---

## 🔹 Usando `span`

```css
.item {
  grid-row: span 2;
}
```

👉 O elemento ocupa **2 linhas**, a partir da posição inicial.

---

## 🧠 Resumindo

- Controla a **posição vertical** no Grid
    
- Usa **linhas horizontais** como referência
    
- Pode definir início/fim ou quantas linhas ocupar
