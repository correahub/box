# repeat() é uma função do CSS Grid que permite repetir um conjunto de tamanhos.
























A função `repeat()` no CSS é usada para **repetir automaticamente um padrão de valores**, geralmente na definição de colunas ou linhas no Grid.

Ela evita a necessidade de escrever o mesmo valor várias vezes, tornando o código mais limpo e fácil de manter.

---

## 💡 Definição direta

`repeat()` é uma função do CSS Grid que permite **repetir um conjunto de tamanhos de forma declarativa**, especificando quantas vezes esse padrão deve aparecer.

---

## 🔹 Exemplo

```css
grid-template-columns: repeat(3, 1fr);
```

👉 Isso é equivalente a:

```css
grid-template-columns: 1fr 1fr 1fr;
```

---

## 🔹 Outro exemplo

```css
grid-template-columns: repeat(2, 100px 1fr);
```

👉 Resultado:

```css
100px 1fr 100px 1fr
```

---

## 🧠 Resumindo

- `repeat()` = repetir valores no Grid
    
- Primeiro argumento = quantidade de repetições
    
- Segundo argumento = padrão a ser repetido
