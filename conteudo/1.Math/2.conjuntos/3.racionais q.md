# O conjunto dos números racionais é formado por todos os números que podem ser escritos como o resultado de uma fração entre dois números inteiros, com denominador diferente de zero.



















### O que são os números decimais?

Os **números decimais** são números que têm uma **vírgula** para mostrar que uma parte dele é **menor que 1**.

Ou seja, não estamos falando só de um número inteiro inteiro como 1, 2, 3…  
Mas também de **pedacinhos** desses números.

---

### 🧁 Imagine um bolo

Você tem **1 bolo inteiro**.  
Esse é o número **1**.

Agora imagine que você **corta esse bolo em 10 fatias iguais**.  
Se você comer **1 fatia**, você não comeu o bolo inteiro, certo?

Você comeu **um pedacinho**.

Esse pedacinho é **0,1** (lê-se _zero vírgula um_).  
Se comer 3 fatias → **0,3**.  
Se comer meio bolo → **0,5**.

Então números como:

- **0,1**
    
- **0,5**
    
- **1,7**
    
- **3,14**
    

São **números decimais**, porque eles mostram **partes** de um todo.

---

### 🧮 Como reconhecer um número decimal?

Ele **tem vírgula** (ou ponto, dependendo do país).  
Exemplos:

- 2,5 → _dois e meio_
    
- 0,25 → _zero vírgula vinte e cinco_
    
- 7,9 → _sete vírgula nove_
    

---

### 🌟 Resumo simples:

**Números decimais são números que têm vírgula e representam partes menores que um inteiro.**

---

### ✅ **Os números decimais fazem parte dos números racionais.**

Ou seja:

> **Todo número decimal é um número racional.**  
> Mas **nem todo racional é decimal** _escrito com vírgula “bonitinha”_.

### Agora o importante:

O **conjunto dos decimais (D)** é **apenas uma parte** dos racionais.

```
        Racionais (ℚ)
       ┌───────────────┐
       │ Decimais (D)  │  ← só os que têm vírgula exata (não infinita)
       └───────────────┘
```

Então:

D⊂Q

Lê-se: **D está dentro de ℚ**.

---

### 📌 Resumo para memorizar:

> **O conjunto dos números decimais (D)** é formado pelos números que têm vírgula **e terminam**, como 1,5 ; 0,75 ; 2,0.  
> **Ele é um subconjunto dos números racionais (ℚ).**
