# O conjunto dos números reais é representado pela união do conjunto dos números racionais com o conjunto dos números irracionais.






















### 🌍 Imagine um “mundo dos números”

Pensa que todos os números que você já conheceu vivem juntos num **grande planeta chamado “Planeta dos Números Reais”** 🌎.  
Esse planeta é o **conjunto dos números reais**, e o símbolo dele é:

👉 **ℝ**

Dentro desse planeta, existem vários **reinos** diferentes, e cada reino tem um tipo de número.

---

### 🏰 1. Reino dos Números Naturais (ℕ)

São os números que usamos para **contar as coisas**:  
👉 0, 1, 2, 3, 4, 5, 6, …

Exemplo:

- “Tenho **3** balas.” 🍬🍬🍬
    
- “Vi **7** cachorros.” 🐶🐶🐶🐶🐶🐶🐶
    

Esses são **naturais**, porque são usados na contagem normal.

---

### 🏕️ 2. Reino dos Números Inteiros (ℤ)

Aqui vivem **os naturais e os negativos também**:  
👉 … -3, -2, -1, 0, 1, 2, 3 …

Exemplo:

- Se eu tinha **3 balas** e perdi **3**, agora tenho **0**.
    
- Se devo **2 reais**, isso é **-2** 💸
    

Esses números podem ser positivos, negativos ou zero.

---

### 🌊 3. Reino dos Números Racionais (ℚ)

Esses são os números que **podem ser escritos como fração**, tipo:  
👉 ½, 0,75, -3/4, 2,5

Exemplo:

- Se você corta uma pizza em **2 pedaços** e come **1**, você comeu **½ pizza** 🍕
    
- Se você tem **R$ 2,50**, isso é **2,5** reais
    

Esses números **podem ser escritos como divisão entre dois inteiros** (por exemplo: ½ = 1 ÷ 2).

---

### 🌈 4. Reino dos Números Irracionais (I)

Aqui vivem os números **que não podem ser escritos como fração**.  
Eles têm **infinitas casas decimais que não se repetem**.

👉 Exemplos:

- **π (pi)** = 3,1415926535…
    
- **√2** = 1,4142135…
    

Esses números **nunca terminam e nunca formam um padrão repetido**!

---

### 🧩 Juntando tudo…

Se a gente juntar **os racionais (ℚ)** e **os irracionais (I)**, temos o **planeta completo** dos **números reais (ℝ)**:

👉 **ℝ = ℚ ∪ I**

Isso quer dizer:

> “O conjunto dos números reais é formado pelos números racionais e irracionais.”

---

### 🎯 Em resumo simples:

|Tipo de número|Símbolo|Exemplos|O que representa|
|---|---|---|---|
|Naturais|ℕ|0, 1, 2, 3, 4...|Contagem|
|Inteiros|ℤ|-3, -2, -1, 0, 1, 2...|Positivos e negativos|
|Racionais|ℚ|½, 0,75, -3/4, 2,5|Números que podem ser frações|
|Irracionais|I|√2, π, e...|Números com dízimas infinitas não repetidas|
|**Reais**|**ℝ**|Todos acima juntos|O “planeta inteiro” dos números|
