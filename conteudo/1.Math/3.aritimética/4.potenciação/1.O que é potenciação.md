# Base elevado ao expoente é igual a potencia


potencia comum é quando bases de valores diferentes resultam em potenciais iguais






















### 🧮 O que é Potenciação?

A **potenciação** é uma forma **mais rápida** de escrever uma **multiplicação repetida** de um mesmo número.

Por exemplo:  
👉 Em vez de escrever **2 × 2 × 2 × 2 × 2**,  
podemos escrever **2⁵** (lê-se “dois elevado à quinta potência”).

---

### ⚙️ Estrutura da Potenciação

Uma **potência** tem **três partes** principais:

![[pot.png]]

🔹 **a** → **base**  
🔹 **b** → **expoente**  
🔹 **c** → **potência** (ou **resultado**)

---

### 🧩 Entendendo cada parte:

- **Base:** é o número que será multiplicado.
    
- **Expoente:** mostra **quantas vezes** a base será multiplicada por ela mesma.
    
- **Potência:** é o **resultado** dessa multiplicação.
    
Base elevado ao expoente é igual a potencia
---

### 💡 Exemplos simples:

#### Exemplo 1:

![[3z.png]]
➡️ ( 3 × 3 = 9 )  
➡️ ( 9 × 3 = 27 )  
➡️ ( 27 × 3 = 81 )

✅ Resultado:  
**3⁴ = 81**

📘 Base = 3  
📘 Expoente = 4  
📘 Potência (resultado) = 81
